## Projetos em produção

### 🛠 Gerador de O.S. — Plataforma Operacional
> Plataforma interna de suporte técnico de um provedor de fibra óptica, com integração completa ao ERP MK Solutions via API REST.

- **37 formulários · 170 variantes** de Ordem de Serviço
- Integração com **8 endpoints REST** do MK Solutions: autenticação, busca de cliente por CPF/CNPJ, criação de protocolo, inserção de comentários (chunking automático) e criação de O.S. vinculada ao cliente
- Reduziu o tempo de atendimento de **15 minutos para menos de 2 minutos**
- Cloud Functions (Node.js 20) + Google Cloud Secret Manager + Firebase
- Desenvolvedor único — front-end e back-end · v4.0.1 · em produção desde **dezembro de 2023**

`React 19` `TypeScript` `Firebase` `Cloud Functions` `Material UI` `REST API` `Secret Manager`

---

### 📸 Eterno Dia — SaaS de Galeria Colaborativa
> Plataforma SaaS para galerias de fotos e vídeos colaborativas em eventos. Convidados enviam mídia via link ou QR Code sem precisar criar conta.

- Integração de pagamentos com **Stripe** (Payment Element, Apple Pay, Google Pay, webhooks)
- 3 planos de assinatura com enforcement real por código (Gratuito · Básico · Premium)
- Galeria em tempo real, crop de imagem no browser, geração de QR Code, download de álbum em ZIP, moderação de conteúdo
- Construído do zero em 6 semanas · Domínio próprio em produção

🔗 [eternodia.com](https://eternodia.com)

`React 19` `Vite` `Tailwind CSS 4` `Firebase` `Stripe` `Cloud Functions` `JSZip`
